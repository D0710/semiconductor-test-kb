# semiconductor-test-kb Skill

## Skill Overview

`semiconductor-test-kb` is a Copilot skill for semiconductor product testing knowledge. It provides answers and guidance based on semiconductor test process documentation, including wafer test, package test, function test, reliability test, and test data analysis.

## Purpose

This skill is designed to help engineering and test teams quickly find information about semiconductor test workflows, test types, test objectives, and key equipment.

## What it contains

- Semiconductor testing process overview
- Wafer test and package test concepts
- Test classification and common test items
- Test equipment and flow descriptions
- Quality and yield analysis guidance

## Usage

Use this skill to ask questions such as:

- "What is the wafer test process?"
- "How do chip packaging tests differ from wafer tests?"
- "What are the key steps in semiconductor test data analysis?"
- "What does CP test mean in semiconductor testing?"
